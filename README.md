# Daily Quote App

A simple web application that displays a new inspirational quote every day.

## Features

- Display a new quote daily
- Clean and simple user interface
- Responsive design
- Easy to customize

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - JavaScript functionality

## Usage

1. Open `index.html` in your web browser
2. View the daily quote
3. Refresh the page to see additional quotes

## Installation

No installation required! Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/aldwindelgado18-ui/daily-quote-app.git
```

- HTML5
- CSS3
