# Daily Quote App

A simple web application and browser extension that displays a new inspirational quote every day.

## Features

- Display a new quote daily
- Clean and simple user interface
- Responsive design
- Easy to customize
- Works as a browser extension on Chrome, Edge, Brave, and more

## Installation

### As a Browser Extension (Chrome, Edge, Brave, Opera)

1. Clone the repository:
   ```bash
   git clone https://github.com/aldwindelgado18-ui/daily-quote-app.git
   ```

2. Open your browser's extension settings:
   - **Chrome/Edge/Brave**: Go to `chrome://extensions/` or `edge://extensions/` or `brave://extensions/`
   - **Opera**: Go to `opera://extensions/`

3. Enable **Developer mode** (toggle in the top-right corner)

4. Click **"Load unpacked"**

5. Navigate to and select the `daily-quote-app` folder

6. The extension will now appear in your browser's extension menu!

### As a Web App

1. Clone the repository or download the files
2. Open `index.html` in your web browser
3. View the daily quote and click "New Quote" for more inspiration

## Files

- `index.html` - Main HTML structure for web version
- `popup.html` - Extension popup interface
- `popup.js` - Extension scripts with Chrome API integration
- `popup.css` - Extension styling
- `style.css` - Web version styling
- `script.js` - Web version scripts
- `manifest.json` - Browser extension configuration
- `icons/` - Extension icons for different sizes

## Usage

### As Extension
1. Click the Daily Quote extension icon in your browser toolbar
2. View the daily quote in the popup
3. Click "New Quote" to get a new inspirational quote

### As Web App
1. Open `index.html` in your web browser
2. Click "New Quote" to refresh and get a new inspirational quote
